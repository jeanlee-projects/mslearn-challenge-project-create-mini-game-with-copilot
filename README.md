# Challenge project - Build a minigame console app with GitHub Copilot

This repository contains a console-based Rock, Paper, Scissors game implemented in `app.py`.

## Game features

- Play against the computer
- Computer randomly chooses rock, paper, or scissors
- Player input is normalized to lowercase
- Invalid options are rejected with a warning
- Player can quit the game at any time by entering `quit` or `q`
- After each round, the player can choose to play again
- Final score is displayed when the game ends

## How to run

From the repository folder, run:

```bash
python app.py
```

## Controls

- Enter `rock`, `paper`, or `scissors` to play a round
- Enter `quit` or `q` to exit immediately
- After a round, enter `yes`/`y` to play again or `no`/`n`/`quit` to stop
